## PrimeCapes

Private 1.12.2 forge base mod for capes. mixin code is skidded from Phobos 1.9.0, just skid if you want


## Info

Build it yourself
<br>
This is how : 

WINDOWS:
<br>
```
./gradlew setupDecompWorkspace
```
```
./gradlew clean
```
```
./gradlew build
```
<br>

MACOS/LINUX:
<br>

```
./gradlew setupDecompWorkspace
```
```
./gradlew clean
```

```
./gradlew build
```
